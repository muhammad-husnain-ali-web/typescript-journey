## What is TypeScript?
- TypeScript is JavaScript with extra features called types
- It helps developers catch errors early
- It is made by Microsoft

## Why companies use TypeScript?
- Fewer bugs
- Easy to manage large code
- Better teamwork
- Better editor support

## Difference between JavaScript and TypeScript
- JavaScript has no type checking
- TypeScript checks types before running code
- JavaScript errors happen at runtime
- TypeScript errors happen at compile time

## How TypeScript code runs in browser
- Browser does not understand TypeScript
- TypeScript is converted to JavaScript
- JavaScript runs in browser
- `TS → JS → Browser
`
