### Why function types matter
- They define what type of arguments a function accepts
- They define what type of value a function returns
- TypeScript prevents calling functions with wrong data

### What void means
- `void` is used when a function returns nothing
- The function only performs an action
- No value is expected after execution

### One error TypeScript caught today
- Passing a string instead of a number to a function
- Returning a wrong type from a function
- TypeScript showed an error before running the code
